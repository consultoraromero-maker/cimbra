# Sitio web CIMBRA — versión inicial

Este paquete contiene una landing page estática lista para revisar, subir a un hosting o convertir a WordPress/Webflow/Wix.

## Archivos
- `index.html`: estructura y contenido del sitio.
- `styles.css`: diseño visual responsive.
- `script.js`: menú mobile.

## Pendientes antes de publicar
1. Confirmar nombre definitivo y validación marcaria.
2. Definir dominio.
3. Reemplazar `contacto@cimbra.com` por el email real.
4. Agregar fotos profesionales o retratos del equipo.
5. Agregar logos/clientes/testimonios cuando estén autorizados.
6. Conectar formulario a Formspree, Tally, HubSpot, Google Forms o CRM.
7. Crear versión inglesa completa si se desea sitio bilingüe con URL `/en`.
